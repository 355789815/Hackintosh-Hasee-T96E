# Hackintosh-Hasee-T96E（clove p95xer/ep）
  这是神舟精盾系列T96E型号的黑苹果EFI仓库-clove p95xer/ep  
    **理论上神舟精盾系列8代标压CPU笔记本均可使用**  
  ### 配置信息如下：
   CPU: Intel i7-8750H  
   内存：16GB  
   核心显卡：UHD630  
   独立显卡：GTX1060（无法驱动  
   声卡：ALC1220  
   硬盘：KINGSTON RBUSNS8154P3256GJ（SSD）   
   无线网卡：博通 BCM94352Z  
   
  ### 系统版本：Big Sur 11.2
